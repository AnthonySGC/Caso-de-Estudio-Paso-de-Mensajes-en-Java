# Sistema de Gestion de Pedidos - Cafeteria Universitaria

## Descripcion
Este proyecto simula la gestion de pedidos en una cafeteria universitaria utilizando Programacion Orientada a Objetos en Java.

El sistema aplica el paso de mensajes entre objetos para representar la comunicacion entre:
- Cliente
- Cajero
- Pedido
- Producto
- Cocina

## Funcionamiento
El flujo del sistema es el siguiente:

1. El cliente realiza un pedido
2. El cajero registra el pedido
3. El cajero agrega productos
4. El cajero envia el pedido a cocina
5. La cocina procesa el pedido
6. El cajero notifica al cliente


## Estructura del proyecto
- Main.java
- Cliente.java
- Cajero.java
- Cocina.java
- Pedido.java
- Producto.java
